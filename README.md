# 黏液橡膠龍頭
黏液科技的附加 繁體翻譯版

黏液橡膠龍頭是一個黏液科技的附加插件, 其靈感來自於著名的模組 [IndustrialCraft 2](https://www.curseforge.com/minecraft/mc-mods/industrial-craft).
它添加了抽取器至黏液科技.
在任何原木上使用抽取器, 你將會獲得橡膠, 並且可以將其轉變成塑料. <br>
這為黏液科技添加了獲得塑料的其他方法.

此附加也增加一些魔法方面, 例如琥珀, 魔法鏡子與其他琥珀相關的物品.
